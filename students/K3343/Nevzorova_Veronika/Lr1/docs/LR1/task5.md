# Задание 1

## Описание

Написать простой веб-сервер для обработки GET и POST HTTP-запросов с помощью библиотеки socket в Python.

## Стек

- Язык: Python
- Библиотека: socket
- Протокол: HTTP

## Как запускать

1. Сервер:
    ```bash
    python3 server.py
    ```

2. Просмотр данных

        Просмотреть в браузере страницу
   ```
   http://127.0.0.1:8058
   ```
