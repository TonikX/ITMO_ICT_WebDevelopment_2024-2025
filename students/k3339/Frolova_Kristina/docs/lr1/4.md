# Задание 4: Многопользовательский чат

## Краткое описание задания

Реализовать двухпользовательский или многопользовательский чат.

## Стек реализации

- Язык: Java
- Пакеты: net, io
- Протокол: TCP

## Как запускать

* Запустите сервер:

```bash
javac Server.class
java Server
```

* Запустите клиент:

```bash
javac Client.class
java Client
```
